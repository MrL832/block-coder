# Block Coder - Retro Logo Turtle 🐢

An ultra-simplified, open-source visual programming tool built specifically for early primary school learners (**Ages 6–8 / Key Stage 1 / Years 1 & 2**). 

This tool introduces the fundamental concepts of directional language, sequencing, algorithms, and geometry through a retro, play-based interface inspired by classic Logo Turtle graphics.

## 🎨 Pedagogical Design Features (Built for Years 1 & 2)

Standard coding platforms (like Scratch) can sometimes overwhelm 6-to-8-year-olds with dense text options and complex nested menus. **Block Coder** strips away the noise:

* **Minimalist Block Palette:** Launches with just 4 essential blocks to teach core sequential logic: `Move Forward`, `Turn Right`, `Change Color`, and `Play Sound`.
* **Visual Directional Cues:** Large, highly colorful, block-snapping interface tailored for young motor-skill development.
* **Turtle Sprite Rotation:** The turtle sprite rotates 90 degrees visually on a 2D canvas whenever a turn block is executed, making left/right tracking intuitive.
* **Instant Geometric Feedback:** The turtle leaves a thick, vibrant line trail behind it, allowing children to immediately see the relationship between their block sequence and geometric patterns.
* **Grid-Based Movement:** Moves in distinct, calculable tiles, allowing students to count out moves mathematically.

## 📥 How to Run Locally / Offline

If your school has strict internet filters or you want to run this application completely offline in the classroom:

1. Click the green **`<> Code`** button at the top right of this GitHub page.
2. Select **Download ZIP** and save it to your computer.
3. Extract (unzip) the folder onto your desktop or a shared network drive.
4. Double-click the **`index.html`** file. It will launch instantly in any modern web browser (Chrome, Edge, Safari) without requiring an internet connection.

## 🛠️ Tech Stack & Contribution

This is a pure frontend, zero-dependency project designed for lightweight deployment:
* **HTML5 Canvas** (for the Turtle animation and trail rendering)
* **Tailwind CSS** via CDN (for a clean, responsive layout)
* **Vanilla JavaScript** (Async/Await loop engine for sequential block execution)
